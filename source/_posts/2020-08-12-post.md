按照几何位置排序，其中 45 代表东北方向，其他方向排序可以自己指定，y 正轴（北）方向为 0 度，顺时针旋转

```
sin(45) * x(centroid($geometry)) + cos(45) * y(centroid($geometry))
```

字段长度不足补 0，3 是长度，0 是不足的左侧补 0

```
lpad("你的字段名", 3, '0')
```
